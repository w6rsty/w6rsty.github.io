XLEN

XLEN代表处理器的基本操作数长度和寄存器宽度

| Arch | XLEN bit | | ----- | -------- | | RV32 | 32 | | RV64 | 64 | | RV128 | 128 | SXLEN是特权模式下的XLEN

WARL

Writ Any Read Legal 表示寄存器可以接受任何值的写入，但读取时返回的值必须是一个合法值

实际行为: 当对寄存器写入一个不合法值时，寄存器可能抛弃不合法的位，或调整为一个合法值，使得读取时能保证得到一个合法值
