4.2标题标签*

<h1>~<h6>一级标题到六级标题 作为标题使用，并依据重要性递减

特点：1.标题文字会变粗变大 2.一个标题独占一行

4.3段落和换行标签*

<p>段落标签</p>一段一组

<br />换行标签 强制换行,使后面的文字换行

段落之间间距较大，换行间距小

4.4文本格式化标签

语义 标签（第一组语义更强烈）
加粗 <strong></strong>或<b></b>
倾斜 <em></em>或<i></i>
删除线 <del></del>或<s></s>
下划线 <ins></ins>或<u></u>

4.5<div>和<span>标签

无语义，一个用来装内容的盒子

<div>表示分割，分区<span>表示跨度，跨距，可以添加id

<div>用来布局，一行只能放一个<div>大盒子

<span>用来布局，一行可以放多个，小盒子

4.6图像标签与路径*

1.图像标签

<img>定义HTML页面中的图像

<img src="图像URL" />单标签

src是<img>标签的必须属性，用于指定图像文件的路径和文件名

属性 属性值 说明
src 图片路径 必须属性
alt 文本 替换文本，图像不能显示的文字
title 文本 提示文本，鼠标放在图片上显示的文字
width 像素 设置图形的宽度
height 像素 设置图像的高度
border 像素 设置图像的边框粗细

图像属性必须写在标签名后，属性之间不分顺序

格式：键值对 key=”value”

2.路径（基础）

1.相对路径

相对于HTML页面的位置

相对路径分类 符号 例
同一级路径 img.gif
下一级路径 / images/img.gif
上一级路径 ../ ../img.gif
2.绝对路径

通常从盘符开始，目录下的绝对位置，或完整的网络地址

4.7超链接标签*

1.语法格式

<a href="跳转目标" target="目标窗口的弹出方式">文本或图像</a>

a是anchor

属性 作用
href 必须属性，指定url地址
target 指定页面的打开方式，_self为默认值,_blank为在新窗口打开

2. 链接分类

1.外部链接

电脑路径也行

2.内部链接

网站内部页面之间的相互链接，直接链接内部页面名称即可

3.空链接

没有确定

链接目标时，用#代替

4.下载链接

如果href地址为一个文件或压缩包，会下载这个文件

5.网页元素链接

网页中的各种元素（文本，图像，表格，音频，视频）都可以添加超链接

6.锚点链接

点击链接快速定位到页面中的某个位置

1.在链接文本的href属性中，设置属性值为 #名字的形式
2.找到目标位置标签，添加一个id属性 = 名字

6.表格标签

1.1表格的主要作用

展示数据

1.2表格的基本语法

1.<table></table>是定义表格的标签

2.<tr></tr>定义表格中的行，必须嵌套在<table></table>中

3.<td></td>定义单元格，必须嵌套在<tr></tr>中

td即table data

1.3表头单元格标签

一般用于第一行或第一列，使表头加粗居中显示

th即table head

1.4表格属性

写在<table>标签中

属性名 属性值 描述
align left，center，right 规定表格相对周围元素的对齐方式
border 1或”” 规定表格单元是否拥有边框，默认为””，表示没有边框
cellpadding 像素值 规定单元边沿与其内容之间的空白，默认1像素
cellspacing 像素值 规定单元格之间的空白，默认2像素
width（height） 像素值或百分比 规定表格的宽度（高度）

1.5表格结构标签

​ 表格很长时，将表格分割成表格头部和表格主体两部分

​ 1.<thead>标签 表格的头部区域

​ 内部必须拥有<tr>标签，一般位于第一行

2. <tbody>标签 表格的主体区域

​ 主要用于存放数据本体

3、以上标签全部放在<table>标签中

1.6合并单元格

1.跨行合并

​ rowspan=”合并单元格的个数”

2.跨列合并

​ colspan=”合并单元格的个数”

3.步骤

​ 1.确定跨行还是跨列

​ 2.找到目标单元格，写上合并方式,如<td colspan="2"><td>

​ 3.删除多余的单元格

8.表单标签

1.主要目的

收集用户信息

2.构成

一个完整的表单由表单域，表单控件，提示信息三部分构成

3.表单域

表单域是一个包含表单元素的区域

<form>标签定义表单域，以实现用户信息的收集与传递

<form>会把它范围内的表单元素信息提交给服务器

语法：

常用属性

属性 属性值 作用
action url地址 用于接收并处理表单数据的服务器程序的url地址
method get/post 用于设置表单数据的提交方式，其取值为get或post
name 名称 用于指定表单的名称，区分同一个页面中的多个表单

4.表单控件

1.input输入表单元素

<input>单标签，标签包含一个type属性

属性值 描述
button 定义可点击按钮（一般用于js启动脚本）
checkbox 定义复选框
file 定义输入字段和”浏览”按钮，供文件上传
hidden 定义隐藏的输入字段
image 定义图像形式的提交按钮
password 定义密码字段，该字段中的字符被掩码
radio 定义单选按钮
reset 定义重置按钮
submit 定义提交按钮，把表单数据发送到服务器
text 定义单行的输入字段，默认宽度为20个字符

单选和复选要名字相同才能实现

属性 属性值 描述

value 规定input元素的值
