Bresenham直线算法

假设坐标轴方向x轴向右，y轴向下，在(x0,y0)和(x1,y1)间绘制一条直线，x1−x0>y1−y0，斜率0<k<1 两点间水平距离为x1−x0，垂直距离为y1−y0

使用截距形式表示为

(1)y=y1−y0x1−x0(x−x0)+y0

但是没有必要去用x计算每一点的y值，只需找到在x在增加到某个值时，会使y增加1

因此可以用y1−y0x1−x0来表示x每增加1时，对y的增加量

  int dy = y1 - y0;
  float e = 0.0f;
  float de = float(dy) / float(dx);
  int y = y0;
  for (int i = x0; i < x1; i++) {
    setPixel(i, y, color);

    }
  }
}

fn line(x0: u32, y0: u32, x1: u32, y1: u32) {
  let dx: u32 = x1 - x0;
  let dy: u32 = y1 - y0;
  let mut e: f32 = 0.0;
  let de: f32 = dy as f32 / dx as f32;
  let mut y = y0;
  for i in x0..x1 {
    setPixel(i, y, color);
    e += de;
    if e.abs() > 0.5 {
      y += 1;
      e -= 1.0;
    }
  }
}
